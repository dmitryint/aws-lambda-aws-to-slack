# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

In short: be respectful, be constructive, assume good intent. Personal attacks, derogatory comments, and discriminatory behavior are not welcome in issues, pull requests, or any other project space.

## Reporting

Report incidents privately to `d.p.karpov@gmail.com` with subject `[conduct] aws-lambda-aws-to-slack`. Reports are kept confidential. Maintainers will review and respond as soon as practical.
