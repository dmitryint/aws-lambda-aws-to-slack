# Security Policy

Please report vulnerabilities **privately**, not via public issues or PRs.

- Open a draft advisory: <https://github.com/dmitryint/aws-lambda-aws-to-slack/security/advisories/new>
- Or email `d.p.karpov@gmail.com` with subject `[security] aws-lambda-aws-to-slack`.

Only the latest `0.1.x` release is supported.
